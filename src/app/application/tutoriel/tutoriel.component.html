Mini tutoriel Angular 10 Site officiel

Définition

Angular est un framework pour le développement d'applications d'entreprises modulaires. Il permet de créer des SPA pour lesquelles les développeurs construises des modules activés par la route.

Conception

L'assemblage d'une page angular se fait avec des composants. Il est aussi nécessaire d'utiliser des interfaces et de définir les routes.

Creation

Pour creér un projet Angular avec la classe de routage, on utilise la commande :
ng new my-app --ng4 --routing
Pour créer une composante :
ng g c my-component
Pour créer une interface :
ng g i my-interface

Mini Tuto TypeScript Site officiel

Qu'est ce que c'est ?

JavaScript est un langage non-typé. TypeScript est un langage par dessus JS qui ajoute des types (annotations, génériques, interfaces) et rapproche le langage à Java, qui et très typé

Comment l'installer ?

Installation via npm :
npm install -g typescript
Rouleur le compilateur via tsc:
npx tsc

*Node Js est un prérequis*
